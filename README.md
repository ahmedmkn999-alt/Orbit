# 🚀 Orbit - Advanced Social Network

أقوى منصة تواصل اجتماعي مع نظام أمان متقدم ومكالمات فيديو بفلاتر!

## ✨ المميزات

### 🔐 أمان عالمي المستوى
- مصادقة Firebase + OTP
- تشفير End-to-End
- نظام حظر متقدم
- تخزين آمن مشفر

### 📞 مكالمات أفضل من Messenger
- فيديو HD مع WebRTC
- 9 فلاتر مباشرة
- مكالمات صوت وفيديو
- تبديل الكاميرا

### 🌍 متعدد اللغات
- عربي وإنجليزي
- تبديل سلس
- RTL/LTR كامل

### 📱 PWA كامل
- يعمل بدون نت
- إشعارات Push
- تثبيت كتطبيق

## 🛠️ التشغيل

1. أنشئ مشروع Firebase
2. انسخ `.env.example` إلى `.env` وضع بياناتك
3. `npm install`
4. `npm run dev`

**صُنع بكل ❤️ في مصر 🇪🇬**
